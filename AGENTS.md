# Project instructions

## Agent skills

### Issue tracker

Project work is tracked as local Markdown files under `.scratch/`. See `docs/agents/issue-tracker.md`.

### Triage labels

Use the five default triage roles. See `docs/agents/triage-labels.md`.

### Domain docs

This repository uses a single-context domain documentation layout. See `docs/agents/domain.md`.

## Working language

- Use Indonesian for collaboration and internal planning unless another language is requested.
- Russian customer-facing copy must be reviewed by a fluent Russian speaker before publication.
- Treat customer names, phone numbers, travel dates, and special requests as sensitive data.

## Project scope

This project explores and builds a Russian-language Bali tour discovery and booking experience, including package presentation, customer enquiries, booking handoff, and operational coordination.
